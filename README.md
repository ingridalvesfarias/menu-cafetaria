# ☕ Menu Digital - Cafetaria & Confeitaria

Projeto de cardápio digital moderno e responsivo, focado em uma experiência limpa para o cliente final. Ideal para estabelecimentos que utilizam QR Code na mesa.

## ✨ Tecnologias
- HTML5 Semântico
- CSS3 (Flexbox e Grid)
- Google Fonts

## 🎨 Estilo Visual
O design utiliza uma paleta de cores creme e marrom café, com tipografia contrastante entre o clássico (*Playfair Display*) e o moderno (*Poppins*).

## 📱 Responsividade
O layout foi construído com a abordagem *mobile-first*, garantindo que a visualização em smartphones seja intuitiva e organizada.
